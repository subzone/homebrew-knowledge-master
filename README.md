# Homebrew Tap for Knowledge Master

## Install

```bash
brew tap subzone/knowledge-master
brew install knowledge-master
```

## Alternative (if formula has issues)

```bash
brew install pipx
pipx install knowledge-master
```

## Usage

```bash
km start
km setup cursor  # auto-configure for your AI tool
km index ~/your-project
km search "how does auth work"
```
